# CLAUDE.md — 道藏知识库 CodeX 工作规范

## 项目身份

道藏经典知识可视化 Demo，参考 `shiji-kb-main` 的管线方法论，处理《道德经》81章与《南华真经》33篇。

## 最高优先级规则

### 标注铁律

1. 标注只添加 `〖TYPE〗` 标记符号，严禁增删改原文字符。
2. 严禁嵌套标注。
3. 去除所有标记后，必须与 `archive/` 底本逐字相同。
4. 异体字保持原貌。

### 实体标注语法

使用 CJK 六角括号 `〖〗` 与单字符前缀：

- `@` 人物
- `?` 神灵
- `=` 地名
- `~` 核心概念
- `{` 经典
- `#` 身份/称谓
- `&` 学派
- `+` 生物/自然物
- `•` 器物
- `$` 数量/度量

### 消歧语法

`〖TYPE 显示名|规范名〗`

### Purple Numbers

- 格式：`[1] [2] [3]`
- 章内从 1 开始
- 一旦分配不可随意变更

## 文件操作规范

- `archive/` 下文件不可手工修改
- `chapter_md/*.tagged.md` 是标注工作文件
- 每次标注后必须运行 `scripts/08_validate.py`

## 数据规范

- 所有 JSON 使用 UTF-8 编码，2 空格缩进
- 实体 ID 格式：`{经典前缀}_{类型}_{序号}`
- 关系 ID 格式：`{source}_{relation}_{target}`
